CSS font-size

font-sizeを使うと文字サイズを変更できます。

クラス名を使うときはcssの部分をstyleタグで囲います。

<style>(入れたいcss)</style>
HTML

<font style="font-size:28px;">文字サイズ28px</font>
CSSとHTML

(css) 
.(クラス){font-size:28px};
(html)
<font class="クラス名">クラスを使ったやり方</font>
例
